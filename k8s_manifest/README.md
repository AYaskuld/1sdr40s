Код ниже отправляет GET запрос на Ip адрес пода на 80 порт, если вернется ответ со статутом 200-399, то под будет считаться healthy.
```bash
startupProbe:
  httpGet:
    path: /
    port: 80
  initialDelaySeconds: 5
  failureThreshold: 40
  periodSeconds: 1
```
